# Blog GitHub

Blog GitHub usando React, Next.js e API do GitHub

## Tecnologias utilizadas
- React
- Next.Js
- API do GitHub

## Passo a passo

1 - Clone este repositório;
<br /> <br />
2 - Instalar pacote de dependências para fazer build do projeto;
<br /> `npm install`
<br /> <br />
3 - Executar/subir projeto;
<br /> `npm run dev`
